## Como capturar

1. Abra `build/index.html` via servidor local (veja README.md).
2. Navegue até cada tela listada acima.
3. Use resolução 1440×900 ou 1280×800 para consistência.
4. Salve os arquivos nesta pasta com os nomes sugeridos.

## Observação

O banner verde "Demonstração interativa" deve aparecer em todas as capturas — isso reforça que se trata de uma demo segura.
