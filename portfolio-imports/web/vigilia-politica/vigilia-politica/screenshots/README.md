## Dicas de captura

- Resolução recomendada: **1440×900** ou **1920×1080**
- Usar tema escuro padrão do sistema
- Garantir que o login já foi feito (sessão ativa)
- Ocultar barra de endereço do navegador na composição do case, se aplicável
