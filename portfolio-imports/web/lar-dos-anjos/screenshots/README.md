## Como capturar

1. Sirva o build: `npx serve build -p 4173`
2. Abra `http://localhost:4173`
3. Em `/entrar`, clique em **Entrar na demonstração**
4. Use resolução 1440×900 ou 1280×800 para consistência no portfólio
